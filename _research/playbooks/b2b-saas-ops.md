# B2B SaaS Ops playbook

Tenant ICP : horizontal B2B SaaS for ops / RevOps / CS / HR / IT — tools that sell to internal-ops teams across industries.

## TAM seed queries

- `industry: "Computer Software" AND keywords: "ops platform" OR "RevOps" OR "workflow automation"`
- `headcount 51-1000` (mid-market — most addressable)
- `funded between $5M-$100M`
- `Series A / B`

## Buying signals (5 canonical)

| Signal | Detection | Why it fires |
|---|---|---|
| **revops_or_ops_lead_hired** | New VP RevOps / Head of Ops / Chief of Staff | First 90-day tooling rationalisation play |
| **funding_growth_round** | Series A or B in last 90d | GTM scaling moment |
| **g2_or_capterra_listing** | New G2 / Capterra listing or category badge | Marketing-led growth → outbound complement |
| **integration_partnership** | Marketplace listing on Salesforce / HubSpot / Zapier | Distribution event |
| **headcount_growth_burst** | ≥20% headcount growth last 6 months | Operational pain point at scale |

## Sequence templates (3)

### Sequence A — "RevOps hire congrats"
```
Step 1 (D+0):
Subject: "<Name>'s first 90 days — playbook"
Body: Recognise the new RevOps / Ops lead. Reference one playbook from a comparable mid-market RevOps role.

Step 2 (D+5):
Subject: "Quick benchmark"
Body: One concrete benchmark (e.g. "comparable companies see X% lift in pipeline efficiency").

Step 3 (D+12):
Subject: "Stepping back"
Body: Long-cycle nurture.
```

### Sequence B — "Stack rationalisation"
```
Step 1 (D+0):
Subject: "Consolidating the stack post-<funding>"
Body: Recognise round + likely rationalisation play. Offer concrete tool-consolidation matrix.

Step 2 (D+6):
Subject: "Re: stack consolidation"
Body: 1-pager.

Step 3 (D+13):
Subject: "Last note"
Body: Soft break-up.
```

### Sequence C — "Marketplace amplifier"
```
Step 1 (D+0):
Subject: "<Marketplace listing> — outbound boost"
Body: Recognise marketplace presence. Outbound pairs well with marketplace inbound.

Step 2 (D+5):
Subject: "Marketplace + outbound playbook"
Body: 1-pager.

Step 3 (D+12):
Subject: "Stepping back"
Body: Soft.
```

## Pipeline stages

1. **Discovery**
2. **Champion + use case**
3. **Trial / POC** — 14-30 days
4. **Procurement** — IT / Security / Finance review
5. **Legal** — DPA + MSA
6. **Won / Lost**

## Common objections

- *"We have HubSpot / Salesforce."* → Position as outbound execution layer ABOVE existing CRM, not replacement.
- *"Onboarding will take a quarter."* → Show 7-phase wizard (≤90 min). Compete on time-to-value.
- *"Our team is too small for a tool."* → Self-serve tier ; founder owns the wizard.

## Geo notes

- US : largest B2B SaaS Ops market. Mid-cycle 4-6 weeks.
- UK : mature, 6-8 weeks.
- DE / NL / DK : structured procurement, 8-12 weeks. Strong fit for outbound automation messaging.
- Skip emerging markets unless founder has a path.
